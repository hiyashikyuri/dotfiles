### 概要
大久保のvimの設定ファイルです

### 使い方
```
git pull
cd dotfiles
# deinをinstall
# https://github.com/Shougo/dein.vim
# deinをinstallしていなくて正常に機能しなかったこともあるので、これ重要
source install.sh
```


### 基本コマンド
```
<C-q + -> 水平ペイン追加
<C-q + |> 垂直ペイン追加
<C-q + space> ペイン入れ替え
<C-e> tree
<C-p> ファイル検索
<C-g> 文字検索
<C-]> 定義元ジャンプ
<,> 左のタブに移動
<.> 右のタブに移動

その他あると思うけど調べてください
